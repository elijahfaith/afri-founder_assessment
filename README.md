Overview
This project is a simple Flutter application built for the Africa Founder Assessment.  
The goal is to demonstrate:

Local storage and CRUD operations using Hive
State management using Riverpod (StateNotifier + Provider)
Persistent data handling across app restarts
Light/Dark theme switching stored locally


 Add Item - User can add an item with title && description 
 Update Item - Modify existing item and save update to Hive 
 Delete Item - Remove item locally by swiping left
Persistent Storage - All data remains even after closing the app 
 Theme Switching - Light/Dark theme using Riverpod + Hive 




Flutter - UI development  
Hive  Local database for persistence 
flutter_riverpod - State management 
uuid - Generate unique IDs for items
